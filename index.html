Loops and Conditions in JS

If condition

In JavaScript, if statements are used to execute a block of code if a certain condition is true. else the control comes out of the block

If else condition

In JavaScript, if-else statements are used to execute a block of code if a certain condition is true. If the condition is false, another block of code can be executed. Here, condition is the expression that is evaluated to either true or false. If condition is true, the code inside the first block will be executed, otherwise the code inside the second block will be executed

If Else-if condition

In JavaScript, the else if statement is used to specify a new condition to test if the first condition is false.

Switch condition

In computer programming languages, a switch statement is a type of selection control mechanism used to allow the value of a variable or expression to change the control flow of program execution via search and map. The switch statement is used to perform different actions based on different conditions. The JavaScript Switch Statement Use the switch statement to select one of many code blocks to be executed.

while loop

A while loop in JavaScript is a loop that executes a block of code as long as a specified condition is true. The while loop loops through a block of code as long as a specified condition is true. If the condition is false, the loop will stop.

For loop

A for loop is a statement that allows you to execute a block of code a number of times. The initialization part is executed once before the loop begins. It is usually used to declare and initialize a counter variable. The condition part is evaluated before each loop iteration. If it evaluates to true, the code block is executed. If it evaluates to false, the loop is terminated
